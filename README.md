## Project Overview

This project focuses on building a classification model to predict the species of iris flowers based on their physical measurements. It demonstrates a complete machine learning workflow, including data exploration, feature analysis, model building, and performance evaluation.

---

## Dataset

The dataset consists of 150 observations with the following features:

* Sepal Length
* Sepal Width
* Petal Length
* Petal Width
* Species (Target Variable)

The dataset is clean, balanced, and widely used for classification tasks.

---


## Tech Stack

* Python
* Pandas, NumPy
* Matplotlib, Seaborn
* Scikit-learn

---

## Workflow

### 1. Data Understanding and Exploration

* Dataset inspection using statistical summaries
* Visualization using pairplots and heatmaps
* Identification of feature relationships

### 2. Feature Analysis

* Petal features show strong class separation
* Sepal features exhibit overlap across species

### 3. Model Building

The following models were implemented:

* Logistic Regression
* K-Nearest Neighbors (KNN)

### 4. Evaluation

* Accuracy score
* Confusion matrix
* Classification report

---

## Key Findings

* Petal length and petal width are the most influential features
* Setosa is linearly separable from other classes
* Versicolor and Virginica exhibit overlapping feature distributions
* Logistic Regression performs efficiently due to clear decision boundaries
* KNN captures local patterns effectively

---

## Model Performance

| Model               | Performance                   |
| ------------------- | ----------------------------- |
| Logistic Regression | High accuracy, interpretable  |
| KNN                 | High accuracy, distance-based |

Logistic Regression is preferred due to simplicity and interpretability.
